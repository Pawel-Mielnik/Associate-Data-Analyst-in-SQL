# Students' Mental Health – SQL Project 01

This repository contains my first SQL mini–project based on the **"Analyzing Students' Mental Health in SQL"** project from DataCamp.

The goal of the project is to explore whether the length of stay in Japan is related to mental health outcomes among international university students, using PostgreSQL for data exploration and aggregation.
